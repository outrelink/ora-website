# Quick Deploy to Vercel

## Deploy Command

```bash
cd /Users/zahra/ORA/ORA-website
vercel --prod
```

Or if you need to install Vercel CLI first:
```bash
npm i -g vercel
vercel login
cd /Users/zahra/ORA/ORA-website
vercel --prod
```

## What Gets Deployed

- ✅ `adaptive-icon.png` - Your ORA logo (will be at https://myora.co/adaptive-icon.png)
- ✅ All email templates
- ✅ Admin dashboard
- ✅ All API functions

## After Deployment

Test the logo URL:
```
https://myora.co/adaptive-icon.png
```

Should show your orange brushstroke icon!
